Sources and methodology

WhereAliens is meant to be source-forward. This page explains where the data and references come from, what each source is useful for, and where caution is needed.

Core sources

These are the primary references planned for the first version of the site. Every section of WhereAliens should clearly name its source and avoid blurring official, curated, and community material together.

  • Use in V1: context, references, source transparency

Methodology principles

WhereAliens should separate data types instead of flattening them together. Official, curated, and community sources each serve different purposes and should be labeled that way.

Attribution rules

Every chart, map result, article, or media reference should visibly state its source, and where relevant, the time range or scope behind it.

Known limitations

Coverage, recency, and classification quality vary by source. The site should show those limits explicitly instead of implying perfect completeness.

WhereAliens is a non-commercial project exploring UFO data, reporting, and media with visible source attribution.
